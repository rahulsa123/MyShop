# MyShop

### MyShop is an online store.

### Technologies

* Python 3.7.
* Django >3.
* Celery for asycronus email.
* rosetta for UI translation.
* parler for model translation.
* WeasyPrint for PDF genration.
        
### this application is avaliable in two language English and hindi
